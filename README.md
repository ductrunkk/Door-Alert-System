# Door Alert System
This project is a simple embedded system that automatically plays an audio notification when a customer enters or exits the store.

## Objective
To solve the need for an audible alert when customers pass through the doorway, using an infrared sensor to trigger a sound module.

## Components Used
- Microcontroller: Arduino Nano V3.0 ATmega328P
- Sensor: E18-D80NK IR Obstacle Sensor
- Audio Module: JQ6500 MP3 Player Module
- Output: 3W 4Ohm Speaker
- Power: 5V 2A Adapter
